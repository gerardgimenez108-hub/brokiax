# Informes de Brokiax

## Estructura

- `auditorias/` revisiones técnicas y de producto
- `estrategia/` posicionamiento, roadmap y decisiones de foco
- `producto/` estado del producto, UX, funcionalidad y operaciones
- `growth/` adquisición, narrativa, contenido y crecimiento
- `operaciones/` mantenimiento, despliegue, checklist y soporte
